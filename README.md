# creator-debugview
一款查看游戏运行时调试信息以及展示log输出信息的插件，对于排查app疑难bug有很大帮助。

cocoscreator 版本：2.2.1

使用方法：

拷贝rich-debug-view到package文件夹

RichDebug.showDebug() 打开调试面板

RichDebug.hideDebug() 关闭调试面板
